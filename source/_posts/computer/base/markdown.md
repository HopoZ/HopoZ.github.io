# 缩进的影响范围

缩进规则在 Markdown 中有以下区别：

    少量缩进：部分解析器会容忍 2 个空格的缩进，并正确解析标题。
    大量缩进（超过 4 个空格或 1 个 Tab）：会被解析为代码块。

Markdown 渲染器更倾向于将 1 个 Tab 的内容识别为代码块。如果想要用空格缩进，可以用`ctl+]`来进行多行空格缩进
