> Problem: [136. 只出现一次的数字](https://leetcode-cn.com/problems/single-number/description/)

# 思路
利用异或运算的性质，两个相同的数异或为0，一个数和0异或为自己。满足交换律和结合律。

# code

```C++
class Solution {
 public:
  int singleNumber(vector<int> &nums) {
    int res = 0;
    for (int i = 0; i < nums.size(); ++i) {
      res ^= nums[i];
    }
    return res;
  }
};
```
