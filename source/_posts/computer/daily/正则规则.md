# WPS业务
```
^(.+?(?:\n|$))
```
匹配第一行内容

```
^[一二三四五六七八九十]{1,}、([^。|\r|：])*[。|\r|：]
```
匹配以中文数字（一、二、三、四、五、六、七、八、九、十）开头，后跟顿号（、），接着是任意数量的非句号、非换行符、非冒号的字符，最后以句号、换行符或冒号结尾的字符串。

```
^（[一二三四五六七八九十]{1,}）([^。|\r|：])*[。|\r|：]
```
匹配以中文括号和中文数字编号开头的文本。

```
^[0-9]{1,}\.([^。|\r]*)[。|\r]
```
匹配以数字编号和英文句点开头的文本。

```
^\d{4}年(0[1-9]|1[0-2])月(0[1-9]|[12][0-9]|3[01])日$
```
匹配 YYYY年MM月DD日 格式
