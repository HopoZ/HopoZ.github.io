Recv failure: Connection was reset错误

  1. 检查防火墙设置：有时防火墙可能会阻止 Git 连接到远程仓库。确保你的防火墙设置允许 Git 通过。你可以尝试暂时禁用防火墙，然后再次尝试 Git 操作，以查看是否仍然出现错误。

  2. 检查代理设置：如果你在使用代理服务器来连接互联网，确保你的代理设置是正确的。你可能需要配置 Git 来使用代理服务器。
