告别鼠标手，尽快选vim. 现在感觉vim单独使用比较好，和IDE的快捷键太容易冲突了，我会在虚拟机上单独vim，虚拟机就是为了从无到有体验编程的存在。

插入

插入—普通模式 在插入模式中使用Ctrl + o 来进入该模式。该模式运行用户暂时回到普通模式，然后在执行一条普通模式的命令之后自动回到插入模式中。

删除

插入模式下删除：
Ctrl + h: 删除光标前一个词(与退格键相同)
Ctrl + w: 删除光标前一个单词
Ctrl + u: 删除至行首
x 快速删除一个字符

替换

移动

gg 光标移动到文件开头
G 光标移动至文件末尾

b 或 B 光标移动至上一个单词的单词首
w 或 W 光标移动至下一个单词的单词首

fx 光标移动至当前行中下一个 x 字符处
Fx 光标移动至当前行中上一个 x 字符处
0 或 Home：光标移动到本行开头

复制粘贴

ggyG复制全文
y复制选中的文本
yy复制当前行
d 剪贴文本
p (小写)新起一行粘贴文本
P 在当前位置粘贴文本
